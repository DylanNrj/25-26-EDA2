# Reto 002 - Criptaritmos

## Autor
Dylan Naranjo

## Problema
Resolver criptaritmos usando recursividad:

- SEND + MORE = MONEY
- FORTY + TEN + TEN = SIXTY
- ODD + ODD = EVEN

## Enfoque
Se utilizó backtracking recursivo:

- asignación de dígitos a letras
- validación por columnas
- control de acarreos
- poda de soluciones inválidas
- patrón hacer / deshacer
