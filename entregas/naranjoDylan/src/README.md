## Descripción
Resolución de criptaritmos mediante dos enfoques:

- Versión recursiva (backtracking clásico)
- Versión iterativa (simulación con pila)

---

## Problemas resueltos

- SEND + MORE = MONEY  
- FORTY + TEN + TEN = SIXTY  
- ODD + ODD = EVEN
